# 待办事项 Todo App

一个简洁的待办事项网页应用，使用原生 HTML / CSS / JavaScript 构建。

## 功能

- 添加待办事项（输入后按回车或点击添加按钮）
- 勾选标记完成 / 取消完成
- 删除单条事项
- 按状态筛选：全部 / 未完成 / 已完成
- 一键清除已完成事项
- 数据自动保存（localStorage），刷新不丢失

## 技术栈

- HTML5
- Tailwind CSS（CDN）
- 原生 JavaScript

## 项目结构

```
todo-app/
├── index.html       # 页面结构
├── css/
│   └── style.css    # 自定义样式
└── js/
    └── app.js       # 业务逻辑
```

## 使用方法

直接用浏览器打开 `index.html` 即可，无需任何构建工具。
